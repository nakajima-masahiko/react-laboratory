# 後藤達也氏の投資スタイル分析

## 概要
後藤達也氏スタイルは、値動きの予想よりも「マクロ文脈の理解」を先に置くアプローチです。特に金融政策（FOMC・日銀）を起点に、景気・物価・企業業績へ連なる因果を重視します。

## 投資哲学の中核
- 価格の前に文脈を読む（インフレ・金利・為替・政策）。
- 中央銀行をハブに、株式・債券・為替を統合して見る。
- 不確実性が高い局面では、強気判断より資金管理を優先する。
- 長期分散・低コスト運用をコアに据える。
- 専門情報を行動に落とせる言葉へ翻訳する。

## 意思決定プロセス（再構成）
1. **マクロ定点観測**
   - `growth` / `inflation` / `policy_rate` / `liquidity` を継続観測。
2. **政策波及の分解**
   - 政策変更が実体経済→企業利益→バリュエーションへどう波及するかを段階整理。
3. **シナリオ確率評価**
   - 強気・中立・弱気の3シナリオを並列化し、確率と観測指標を紐づける。
4. **行動の段階化**
   - 様子見／小幅リバランス／通常積立／リスク圧縮を確信度で選ぶ。
5. **平易化制約**
   - 専門用語は一文で平易化し、実行可能な判断材料に変換。

## 出力テンプレート（実務利用向け）
- 今週のマクロ要約（3行）
- 注目イベント（政策会合・物価統計・雇用）
- 3シナリオ（確率・トリガー・資産別影響）
- 推奨アクション（長期資産と短期資金を分離）
- 想定リスクと見直し条件

## リスク管理上の特徴
- 単一指標で結論を出さない。
- 事実と予測を明示的に分離する。
- 地政学・政治イベントは「方向予想」ではなく「変動レンジ拡大リスク」として扱う。

## 向いている投資家像
- 経済ニュースを投資判断に接続したい人。
- 短期売買より資産形成の再現性を重視する人。
- 相場急変時にルールで落ち着いて行動したい人。
