# テスタ氏の投資スタイル分析

## 概要
テスタ氏スタイルは、短期〜中期の値幅獲得を狙いながらも、最優先事項を「損失限定」と「再現性」に置くアプローチです。攻める場面と守る場面を明確に分け、資金配分を機械的に運用します。

## 投資哲学の中核
- エントリー前に損失上限（1トレード・1日）を固定する。
- 「買う理由」と「売る条件」を同時設計する。
- 地合い（レジーム）でポジションサイズを切り替える。
- 本丸資金とテンバガー探索資金を分離する。
- 感情評価ではなく、取引ログで意思決定品質を改善する。

## 意思決定プロセス（再構成）
1. **リスク予算先行**
   - `max_loss_per_trade` / `max_daily_drawdown` / `max_gross_exposure` を先に確定。
   - 制約超過の提案は却下。
2. **仮説と撤退条件のペア化**
   - すべての提案で `entry_thesis` と `invalidated_if` を必須化。
3. **レジーム適応サイズ**
   - 上昇・レンジ・急落の環境判定で同銘柄でも推奨サイズを可変化。
4. **バーベル配分**
   - `core`（安定）と `explore`（高成長）を分離。
   - テンバガーは `explore` 限定で、失敗前提分散を実施。
5. **事後学習ループ**
   - 取引後にシナリオ妥当性・実行ミス・環境ノイズを分解して再学習。

## 出力テンプレート（実務利用向け）
- 市場認識（地合い）
- 候補銘柄と根拠（3点以内）
- エントリー条件
- 利確／撤退条件
- 推奨サイズ（%）
- 想定最大損失（円・%）
- 代替シナリオ（外れた場合）

## リスク管理上の特徴
- レバレッジ提案は損失耐性・撤退条件の先出しが前提。
- 「確実」などの断定表現を禁止。
- 単一銘柄への過度集中を防ぎ、探索枠にも上限を設定。

## 向いている投資家像
- ルールベースで短中期売買を行いたい人。
- 損失管理を最重要KPIとして運用したい人。
- 成長株の上振れを狙いつつ、ポートフォリオ全体の破綻確率を抑えたい人。
